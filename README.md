## YOLOv8 Live Object Detection (Android)

### Description
This Android application performs live object detection using a YOLOv8 TensorFlow Lite model.

### Getting Started
1. Clone or download this repository.
2. Put your `.tflite` model and `.txt` labels file inside the `app/src/main/assets` folder.
3. Update `MODEL_PATH` and `LABELS_PATH` in `Constants.kt`.
4. Build and run the app.
